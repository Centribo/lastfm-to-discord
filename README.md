# lastfm-to-discord
Set your Discord status to be what you're listening on Last.fm.

## Usage
**Install Prequisites**
```bash
$ cd lastfm-to-discord
$ npm install
```

***Compile Typescript***
```bash
$ npm run compile
```

***Start Service***
```bash
$ npm run start
OR
$ npm start
```

***Compile & Run***
```bash
$ npm run run
```
